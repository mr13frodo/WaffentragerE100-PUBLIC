WaffenträgerE100 v.2.6



СКАЧАННЫЙ РЕПОЗИТОРИЙ ИЗМЕНЯТЬ НИ В КОЕМ СЛУЧАЕ НЕЛЬЗЯ. ДЛЯ ДАЛЬНЕЙШЕГО ИСПОЛЬЗОВАНИЯ ЕГО ЛИШЬ КОПИРОВАТЬ, ИНАЧЕ СЛОМАЕТСЯ МЕХАНИЗМ ОБНОВЛЕНИЯ



Программа перед началом работы с аккаунтом авторизируется в него, для чего потребуется ввести имя сессии, api_id, api_hash и номер телефона аккаунта. На введенный телеграм придет оповещение с кодом, который потребуется ввести. И при наличии двухфакторки, ввести надо будет еще и пароль от аккаунта. Дання операция будет проводиться лишь один раз и в дальнейшем вся информация будет подтягиваться из файлов *имя сессии*.session и *имя сессии*.config


Имеются четыри варианта выбора в главном меню: 
	1 - Работа с аккаунтом
	2 - Работа с чатами
	3 - Работа с рассылкой
	4 - Спам по чатам



РАБОТА С АККАУНТОМ

1 - Прическа аккаунта
Для корректной работы нужна папка base, которая внутри себя содержит:
- Папку avatars. Внутри папки должны находиться изображения, одно из которых будет случайным образом поставлено на аватар аккаунта
- Текстовый файл first_name, который внутри себя содержит строки с именами, одно из которых будет случайным образом выбрано для имени аккаунта
- Текстовый файл last_name, который внутри себя содержит строки с фамилиями, одна из которых будет случайным образом выбрана для фамилии аккаунта
- Текстовый файл status, который внутри себя содержит строки с био, одно из которых будет случайным образом выбрано для био аккаунта
После загрузки и установки всех данных, программа настроит политику конфиденциальности аккаунта по таким параметрам:
- Номер телефоне виден Никому
- Пересылать сообщения разрешено Никому
- Звонить разрешено Никому
- Добавлять в чаты разрешено Никому
Так же создаются папки с:
- Контактами
- Группами
- Каналами

2 - Чистка аккаунта
Содержит 4 варианта чистки аккаунта, каждый из которых удаляет все чаты соответсвенного типа:
- Чистка диалогов
- Чистка групп
- Чистка каналов
- Чистка ботов

3 - Прозвон системных оповещений
Вывод в консоль всех входящих сообщений в диалоге с системой телеграма. Использовать, например, для получения кода для входа.


РАБОТА С ЧАТАМИ

1 - Вступление в чаты
Автоматизированное вступление в чаты из списка. Список берется из файла groups.txt. Вступление в чаты происходит постепенно и требует длительной работы. Вступит во все чаты из файла. Рекомендуется не делать список больше, чем 150 чатов

2 - Предварительная чистка чатов
Покидает все чаты, где непрочитанных сообщений меньше, чем введенное число

3 - Создание папки со всеми чатами
Создает папки со всеми группами, в которых состоит аккаунт. Необходимо ввести название создаваемых папок и количество чатов в каждой папке.

4 - Расширенное вступление в чаты
Автоматизированное вступление в чаты из списка с предварительной чисткой неактивных чатов. Список берется из файла groups.txt. Вступление в чаты происходит постепенно и требует особенно длительной работы. Вступает до тех пор, пока количество чатов после чистки не достигнет указанного вручную значения. Обработанные чаты удаляются из файла groups.txt

5 - Парс участников чатов
Парсит всех участников из чатов, в которых состояит аккаунт. Автоматически фильтрует всех, кто не был давно в сети, администратор, ботов и участников с премиумом.


РАБОТА С РАССЫЛКОЙ

1 - Спам по каналам
Отправка сообщений из избранного в комментарии каналов, на которые подписан аккаунт. Есть 2 режиме работы, один из которых мониторит новые посты в канале и пытается прокомментировать их, а второй комментирует в каждом канале N количество последних постов (вводится вручную). Автоматически рандомизирует буквы в тексте и поддерживается его кастомизация через {|}



СПАМ ПО ЧАТАМ
Отправка сообщений из избранного во все чаты по кругу. Сообщение из избранного считывается перед каждой отправкой, а список чатов обновляется перед каждым новым кругом спама. Автоматически рандомизирует буквы в тексте и поддерживается его кастомизация через {|}. Содержит внутри себя уникальные настройки:
- Минимальное количество сообщений перед отправкой. Смотрит в последних N сообщениях чата, является ли спамер автором. Если да, то чат игнорируется
- Количество участников для тегера. Игнорируется, если введен 0. Если значение отлично от нуля, то тегает в сообщении N участников чата, после чего сразу же редактирует сообщение.
- Текст для автоответчика на первое входящее от пользователя сообщение.
