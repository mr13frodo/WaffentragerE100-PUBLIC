Программа перед началом работы с аккаунтом авторизируется в него, для чего потребуется ввести имя сессии, api_id, api_hash и номер телефона аккаунта. На введенный телеграм придет оповещение с кодом, который потребуется ввести. И при наличии двухфакторки, ввести надо будет еще и пароль от аккаунта. Дання операция будет проводиться лишь один раз и в дальнейшем вся информация будет подтягиваться из файлов *имя сессии*.session и *имя сессии*.config

Имеются три режима работы.

1. Заполнение аккаунта
Для корректной работы нужна папка base, которая внутри себя содержит:
- Папку avatars. Внутри папки должны находиться изображения, одно из которых будет случайным образом поставлено на аватар аккаунта
- Текстовый файл first_name, который внутри себя содержит строки с именами, одно из которых будет случайным образом выбрано для имени аккаунта
- Текстовый файл last_name, который внутри себя содержит строки с фамилиями, одна из которых будет случайным образом выбрана для фамилии аккаунта
- Текстовый файл status, который внутри себя содержит строки с био, одно из которых будет случайным образом выбрано для био аккаунта
После загрузки и установки всех данных, программа настроит политику конфиденциальности аккаунта по таким параметрам:
- Номер телефоне виден Никому
- Пересылать сообщения разрешено Никому
- Звонить разрешено Никому
- Добавлять в чаты разрешено Никому
Так же создаются папки с:
- Контактами
- Группами
- Каналами

2. Заполнения чатами
Внутри себя содержит 3 режима:
- Вступление в чаты. Автоматизированное вступление в чаты из списка. Список берется из файла groups.txt. Вступление в чаты происходит постепенно и требует длительной работы. Вступит во все чаты из файла. Рекомендуется не делать список больше, чем 150 чатов
- Предварительная чистка чатов. Покидает все чаты, где непрочитанных сообщений меньше, чем введенное число
- Создание папки со всеми чатами. Создает папку со всеми группами, в которых состоит аккаунт. Необходимо ввести название создаваемой папки
- Расширенное вступление в чаты. Автоматизированное вступление в чаты из списка с предварительной чисткой неактивных чатов. Список берется из файла groups.txt. Вступление в чаты происходит постепенно и требует особенно длительной работы. Вступает до тех пор, пока количество чатов после чистки не достигнет указанного вручную значения. Обработанные чаты удаляются из файла groups.txt

3. Спам по чатам
Отправка сообщений из избранного во все чаты по кругу. Сообщение из избранного считывается перед каждой отправкой, а список чатов обновляется перед каждым новым кругом спама. Автоматически рандомизирует буквы в тексте и поддерживается его кастомизация через {|}

4. Спам по каналам (ТЕСТ)
Отправка сообщений из избранного в комментарии каналов, на которые подписан аккаунт. Есть 2 режиме работы, один из которых мониторит новые посты в канале и пытается прокомментировать их, а второй комментирует в каждом канале N количество последних постов (вводится вручную). Автоматически рандомизирует буквы в тексте и поддерживается его кастомизация через {|}
